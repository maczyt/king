# Data 处理



 
 
 
 
 
 







const vm = new King({
  el: ".app",
  data() {
    return {
      name: "zyt",
      age: 25
    };
  },
  computed: {
    user() {
      return `My name is ${this.name}, age is ${this.age}`;
    }
  }
});

上面高亮处即是我们需要处理的data数据，怎么处理呢？要处理成怎么样呢？

在 vue 中，当我们修改数据，页面即会发生更新，并且页面也可以使用 input 通过 v-model 来修改 data 所以我们需要 data 数据处理成能检测变化的(也称可观察的)
